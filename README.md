<h1 align="center">MoreBroadcast Plugin</h1>
<p align="center" style="color: #888; font-size: 18px;">
  MoreBroadcast is a simple EXILED plugin that provides two features: automated broadcasts for things like server advertisements, and a welcome message when a player joins the server for the first time.
</p>

<h2>Installation</h2>

1. Download the latest release from GitHub.  
2. Place the DLL inside your Exiled `/Plugins` folder.  
3. Restart your server.  
4. Configure MoreBroadcast using the generated config files.
